# CALCULADORA DE EQUAÇÕES DE 1 E 2 GRAU

A aplicação foi escrita em python.


# Aplicação

```sh
----------------------------------------------------------------------
1 - equação do 1º Grau
2 - equação do 2º Grau
3 - Sair
----------------------------------------------------------------------
Digite a opção desejada:1
----------------------------------------------------------------------


Equação de 1º Grau:
ex.  2x + 4 = 0 => (A = 2 e B = 4)

Digite A:2
Digite B:4

Equação:  2 x+ 4  = 0
X:  -2.0

----------------------------------------------------------------------
1 - equação do 1º Grau
2 - equação do 2º Grau
3 - Sair
----------------------------------------------------------------------
Digite a opção desejada:2
----------------------------------------------------------------------


Equação de 2º Grau:
ex.  4x² - 3x + 6 = 0 => (A = 4 , B = -3 e C = 6)

Digite A:4
Digite B:-3
Digite C:6

Equação:  4 X² (+)  -3 X (+)  6  = 0
Delta:  -87
Raiz:  9.32
x1:  1.54
x2:  -0.79
```
